## Основные

1. `Space + w` - Save file
2. `Space + q` - Quit
3. `jk` - Exit insert mode

## Навигация

1. `Space + Space` - Find file
2. `Space + /` - Search in project
3. `Space + r` - Recent files
4. `Ctrl + h` - Move to left split
5. `Ctrl + l` - Move to right split
6. `Ctrl + j` - Move to bottom split
7. `Ctrl + k` - Move to top split
8. `Shift + h` - Previous buffer
9. `Shift + l` - Next buffer

## Neo-tree

1. `Space + e` - Toggle file explorer
2. `a` - Create file
3. `A` - Create folder
4. `r` - Rename
5. `d` - Delete
6. `c` - Copy
7. `m` - Move
8. `H` - Toggle hidden files

## LSP

1. `gd` - Go to definition
2. `gr` - Show references
3. `K` - Show documentation
4. `Space + ca` - Code actions
5. `Space + cr` - Rename variable
6. `Space + cd` - Show line diagnostics
7. `] + d` - Next error
8. `[ + d` - Previous error

## Git

1. `Space + gg` - Open Lazygit
2. `Space + gb` - Git blame
3. `] + h` - Next hunk
4. `[ + h` - Previous hunk
5. `Space + ghs` - Stage hunk
6. `Space + ghr` - Reset hunk
7. `Space + ghp` - Preview hunk

## Lazygit (внутри UI)

1. `Space` - Stage / unstage file
2. `c` - Commit
3. `p` - Push
4. `P` - Pull
5. `b` - Branches
6. `q` - Close

## Vim основы

1. `i` - Insert mode
2. `v` - Visual mode
3. `V` - Visual line mode
4. `yy` - Copy line
5. `dd` - Delete line
6. `p` - Paste
7. `u` - Undo
8. `Ctrl + r` - Redo
9. `gg` - Go to top
10. `G` - Go to bottom
11. `zz` - Center screen
